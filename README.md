```
vhls-check - Vitis HLS checker

Usage:
  vhls-check [SOURCE_DIR] [options]
  vhls-check --help | -h
  vhls-check --version | -v

Options:
  --help, -h
    Show this help

  --version, -v
    Show version number

  --output, -o OUTPUT
    Output summary to the specified file

  --work-dir, -w WORK_DIR
    Working directory
    Default: ./work

  --docker, -d
    Run in docker container environment

  --force, -f
    Force output even if files or directories already exist

Arguments:
  SOURCE_DIR
    Target source directory
    Default: ./
```
